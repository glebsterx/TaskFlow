# Message Parsing Specification

Цель: определить является ли сообщение задачей.

## Критерии кандидата

Сообщение содержит:
- глагол действия
- срок
- обращение к человеку

## Поддерживаемые паттерны

"сделай ... до ..."
"нужно ... к ..."
"посмотри ..."
"берёшь ...?"

## Confidence Score

0.0 — не задача  
1.0 — высокая вероятность

Порог сохранения: 0.6
